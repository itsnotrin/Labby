In QBittorrent.swift, find the `applyLimits(dlKiBps:ulKiBps:toggleAltMode:)` method (~line 432).

Current broken logic:
```swift
if let _ = toggleAltMode {
    try await client.toggleAlternativeSpeedLimits()
}
```
This toggles alt-mode whenever `toggleAltMode` is non-nil (both `true` and `false` trigger it).

Fix:
Change the condition to only toggle when the value is explicitly `true`:
```swift
if toggleAltMode == true {
    try await client.toggleAlternativeSpeedLimits()
}
```

This uses Optional's `== true` which returns false for both `nil` and `.some(false)`, only matching `.some(true)`.
